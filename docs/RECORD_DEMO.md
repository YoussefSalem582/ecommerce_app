# Recording the demo GIF

The README references `./docs/demo.gif`. To add it:

1. Run the app in demo mode: `flutter run` (default `APP_ENV=demo`).
2. Sign in via **Google Sign-In** (showcase stub) or demo credentials.
3. Record a flow: home grid → product detail → add to cart → cart → checkout → order success.
4. Export as GIF (e.g. ScreenToGif on Windows, LICEcap, or `ffmpeg`).
5. Save to `docs/demo.gif` and commit, or host externally and update the README image URL.

Recommended resolution: 1280×720 or device-native width, ≤15 MB for GitHub rendering.
