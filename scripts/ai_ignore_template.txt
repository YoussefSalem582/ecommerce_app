# ai_ignore_template.txt — Canonical exclusion list for all AI-agent ignore files.
#
# This is the SINGLE SOURCE OF TRUTH for what every AI agent (Cursor, Claude,
# Codex, Copilot, Gemini, Aider, Windsurf, generic) excludes from its context
# window.
#
# To add or remove an exclusion:
#   1. Edit this file.
#   2. Run `.\scripts\sync_ai_ignores.ps1` (Windows) or `bash scripts/sync_ai_ignores.sh` (Unix).
#   3. CI will verify on every PR via `.github/workflows/docs.yml` and the
#      `--check` mode of the sync script.
#
# This file is NOT consumed by any agent directly — only `sync_ai_ignores.{ps1,sh}`
# reads it. Each agent reads its own `.{agent}ignore` file at the project root.
#
# `.gitignore` is intentionally NOT in scope here — it is git-native, not AI-specific.
# ============================================================
