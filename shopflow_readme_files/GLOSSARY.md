# Glossary

| Term | Meaning |
|------|---------|
| ShopFlow | This Flutter app (`shop_flow` package) |
| BLoC | `flutter_bloc` state container for features |
| Cubit | Simpler bloc for wishlist, theme, language, connectivity |
| Either | `dartz` `Left(Failure)` / `Right(success)` |
| Hive | Local NoSQL cache for catalog/cart/orders |
| Dio | HTTP client wrapper in `DioClient` |
| AppRoutes | Typed GoRouter path constants |
| AppPalette | `ThemeExtension` semantic colors |
| Fake Store | Default demo REST API for products/auth |
