# Testing

- Unit tests: `test/` with `package:flutter_test` / `package:test`
- Run: `flutter test`
- Coverage: skill `dart-collect-coverage`
- Widget tests: skill `flutter-add-widget-test`
- Integration: skill `flutter-add-integration-test`, `integration_test/`

Prefer testing domain/use cases and repository mapping without full widget trees when possible.
